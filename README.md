# observables-over-boundary
Allow observables to be communicated over a boundary (such as network)


# Protocol

```
[ Host ]                                     [ Remote ]
       |                                     |
  [1]  |             +--------+              |
  [2]  | ~~~~~~~~~~~~| Obs.Id |~~~~~~~~~~~~> |
       |             +--------+              | [3]
       |                                     |
------------------------------------------------------
       |                                     |
       |           +-----+--------+          |
       | <---------| SUB | Obs.Id |--------- | [4]
       |           +-----+--------+          |
       |    +--------+--------+--------+     |
  [5]  | ---| SUBBED | Obs.Id | Sub.Id |---> |
       |    +--------+--------+--------+     |
       |                                     |
       |      +------+--------+------+       |
  [6]  | -----| DATA | Sub.Id | Data |-----> |
       |      +------+--------+------+       |
       |                                     |
       |     +-------+--------+--------+     |
       | <---| UNSUB | Obs.Id | Sub.Id |---- | [7]
       |     +-------+--------+--------+     |
```
